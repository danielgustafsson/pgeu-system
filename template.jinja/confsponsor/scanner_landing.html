Badge Scanning - {{conference}}

You are registered as a badge scanner for {{sponsor}}.

To scan a badge you will use your regular smartphone. The system has been tested with the most common models of smartphones, and should hopefully work with the one you have.

Native app

For Android there is a native app available in Google Play.

Web app

For both Android and iOS, there is a web app available that can also be used for the scanning. Note that you will need to grant the website permissions to use your camera, and depending on smartphone model you may need to configure the camera before you scan (for example so that it doesn't use the front-facing camera, which some models do by default).

NOTE! Due to an unknown change in a recent update to iOS, when using the web app from an iOS device such as an iPhone, the device must be held in landscape mode instead of portrait for the scan to complete correctly.

Native scanning on device

Finally, you can also scan the attendee badges directly with your browser in the smartphone. This will work around the issue with landscape mode on iPhones, but unlike the other methos to scan the badges you must be logged in with the same account you used for your registration in order to access them. If you choose to use this method, you scan the code directly and do not use the app link below.
